# Get-Artist

## SYNOPSIS
Gets an artist.

## SYNTAX

```
Get-Artist [[-Name] <String[]>] [<CommonParameters>]
```

## DESCRIPTION
Gets an artist.

## EXAMPLES

### Example 1
```powershell
Get-Artist
```

Gets all the artists.

### Example 2
```powershell
Get-Artist Insomnium
```

Gets the artist "Insomnium".
